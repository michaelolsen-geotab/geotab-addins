# Geotab Add-Ins

Custom MyGeotab add-ins for Geotab customers.

| Add-In | Description | Manifest |
|--------|-------------|---------|
| [holt-daily-report](holt-daily-report/) | Daily fleet report: usage hours, miles, fuel, EV energy, fuel/hr | [manifest.json](holt-daily-report/manifest.json) |
| [trips-history-enhancer](trips-history-enhancer/) | Adds address search to the trips history map and restores second-precision stop timestamps | [manifest.json](trips-history-enhancer/manifest.json) |
